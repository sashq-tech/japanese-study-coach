Privacy

Local-first by design.

Japan Ready Coach currently runs in the browser with no account system, backend database, analytics, or external API calls.

What Is Stored

The app stores progress in browser local storage: scores, streaks, kana mastery, weak-item review, selected study focus, mini-session summaries, timer history, and native-speaker review notes.

Where It Stays

That data stays in the browser and device where it was created. It is not sent to a server by this static prototype. Clearing browser storage can delete progress.

Your Controls

The app includes export, import, and reset controls for local progress. Export a backup before switching browsers, clearing cache, or moving to another device.

Future sync or accounts would require a new privacy review and updated copy. Return to app
