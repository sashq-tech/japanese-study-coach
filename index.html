About this project

Built for a real move to Japan.

This project is my study coach for rebuilding Japanese from the beginning: kana first, then N5 vocabulary, particles, grammar, and practical phrases. It is shaped by my Japan background, Emi's current perspective, and the need to keep learning useful enough to continue.

Privacy

No account. No server. No tracking.

Progress, study timers, review items, and notes stay in this browser's local storage. Export a backup before switching browsers or devices. A future hosted version should keep this privacy-first model unless a sync feature is clearly introduced.

Learning scope

Beta learning tool, not an official JLPT claim.

The current focus is beginner-safe N5 practice and habit-building. Kanji, advanced levels, listening, and full JLPT readiness checks are roadmap items. Human review is still needed for naturalness and authenticity.

Travel readiness

Start with money basics.

Planning a Japan move or first trip also means handling small real-world friction. I started the blog with how I would think about cash, cards, and Suica in Japan.

Local data

Everything is stored in this browser only: scores, kana mastery, weak review, study time, mini-session history, and notes.

  • No account, server, analytics, or cloud sync.
  • Export a backup before switching browsers or clearing cache.
  • Reset only when you want a clean local start.

Human review notes

Use this local note pad for feedback from Emi or another reviewer on naturalness, politeness, or better phrase choices. Notes stay in this browser and are included in backup exports.
